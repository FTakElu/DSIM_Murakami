Configuração de Certificado SSL

Configure o certificado SSL para usar o sistema

Importante: O sistema usa um certificado SSL autoassinado. É necessário aceitar o certificado antes de fazer login.

1

Abrir página do backend

Clique no botão "Aceitar Certificado" abaixo para abrir a página do backend em uma nova aba.

2

Aceitar o aviso de segurança

Na nova aba, você verá um aviso de segurança. Clique em "Avançado" ou "Advanced".

3

Prosseguir para o site

Clique em "Prosseguir para 98.92.80.193 (não seguro)" ou "Proceed to 98.92.80.193 (unsafe)".

4

Voltar e fazer login

Feche a aba e volte aqui. Clique em "Ir para o Login" para começar a usar o sistema.
